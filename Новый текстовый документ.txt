    <h2>Меню «Файл»</h2>
    <ul>
        <li><b>Создать</b> — создаёт новый пустой документ с именем <code>Новый файл_N</code>.</li>
        <li><b>Открыть</b> — открывает существующий текстовый файл и загружает его в редактор.</li>
        <li><b>Сохранить</b> — сохраняет текущий документ в тот же файл (если файл уже был сохранён).</li>
        <li><b>Сохранить как…</b> — сохраняет текущий документ в новый файл с выбранным именем.</li>
        <li><b>Выход</b> — завершает работу программы.</li>
    </ul>

    <h2>Меню «Правка»</h2>
    <ul>
        <li><b>Отменить</b> — отменяет последнее действие.</li>
        <li><b>Повторить</b> — повторяет отменённое действие (если возможно).</li>
        <li><b>Вырезать</b> — вырезает выделенный текст в буфер обмена.</li>
        <li><b>Копировать</b> — копирует выделенный текст в буфер обмена.</li>
        <li><b>Вставить</b> — вставляет текст из буфера обмена в текущую позицию курсора.</li>
        <li><b>Удалить</b> — удаляет выделенный текст без помещения в буфер обмена.</li>
        <li><b>Выделить всё</b> — выделяет весь текст в текущем документе.</li>
    </ul>
